RSS 기본 템플릿 앱 시작하는 방법 RSS 기본 템플릿 앱

본 앱은 탭과 사이드 메뉴, 그리고 RSS 를 포함하는 기본 앱입니다. 만약, 새로운 앱을 만들려고 한다면, 이 앱의 github master branch 를 새로운 branch 로 만들어서 작업하면 됩니다.

각 branch 를 작업 할 때, branch-name-release 와 같이 끝애 `-release` 가 붙으면 그 branch 는 실제로 릴리스가 되는 branch 입니다. 따라서 작업이 끝나면, `branch-name-release` 로 push 하시면 됩니다.
